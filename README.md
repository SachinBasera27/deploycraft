# DeployCraft
Hands-on project exploring backend development and DevOps, including Python APIs, PostgreSQL, Azure App Service, CI/CD automation, and application networking.


#SQL DATA:

Database Used: PostgreSQL

Dataset for the Database has been sourced from U.S. Department of Education's College Scorecard program. It contains detailed data on recent college graduates, broken down by their specific field of study 

URL: https://collegescorecard.ed.gov/data/
